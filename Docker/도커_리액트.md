# Docker 컨테이너
- 도커를 이용해 리액트를 실행하는 프로젝트를 만들어 보자

## 1. 노드 설치
- 우선 node.js를 설치한다 
    ```
    brew install node

    node --version

    npm --version
    ```
    - mac 환경에서 brew를 통해 node를 설치한 후 정상적으로 설치 되었는지 확인

## 2. 리액트 설치
- 이후 리액트를 설치한다
    ```
    npx create-react-app ./

    npm run start
    ```
    - 현재 디렉토리에 리액트를 설치 후 리액트를 실행하여 3000번 포트로 접속하여 확인한다.
